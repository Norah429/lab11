# Credit Card Customer Segmentation using K-Means

## Project Description

This project uses K-Means clustering to divide credit card customers into different groups based on their spending behavior.

The goal is to identify customer segments and understand customer patterns.

This is an Unsupervised Machine Learning project because there is no target variable.

---

## Dataset

Dataset: CC_GENERAL.csv

The dataset contains information about:

- Balance
- Purchases
- Cash Advance
- Credit Limit
- Payments
- Purchase Frequency

Total records:

- 8950 customers
- 18 features

---

## Steps Performed

1. Imported required libraries

2. Loaded the dataset

3. Removed CUST_ID column

4. Checked missing values

5. Filled missing values using mean imputation

6. Performed Exploratory Data Analysis

7. Applied feature scaling

8. Used Elbow Method

9. Calculated Silhouette Scores

10. Applied K-Means clustering

11. Analyzed clusters

12. Visualized clusters using PCA

---

## Model Used

K-Means Clustering

Selected number of clusters:

K = 4
